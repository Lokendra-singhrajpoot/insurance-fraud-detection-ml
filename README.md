# Insurance Fraud Detection using Machine Learning

## Overview
Insurance fraud is a major issue in the insurance industry. This project uses machine learning to identify fraudulent insurance claims.

## Objective
To build a classification model that predicts whether an insurance claim is fraudulent or legitimate.

## Technologies Used
- Python
- Machine Learning
- Scikit-learn
- Pandas

## Dataset Features
- Age
- Claim Amount
- Vehicle Age
- Previous Claims

## Model Used
Logistic Regression

## Steps
1. Data preprocessing
2. Train-test split
3. Model training
4. Prediction
5. Accuracy evaluation

## Output
The model predicts whether a claim is fraud or not.
